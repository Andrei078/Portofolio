Test Your Entire Setup — In One Place

A modern web application for testing all your desk peripherals. Keyboard, mouse, controller, audio, and webcam — real-time diagnostics with live stats, latency tracking, drift detection, and audio channel isolation. All in the browser. No installs needed.

// PERIPHERALS

What You Can Test

Five dedicated testing modules — each built around real browser APIs for accurate, live results.

01

Keyboard Tester

Press any key and watch it light up in real time. Detects every keypress, measures input latency in milliseconds, and identifies stuck or unresponsive keys. Perfect for testing mechanical switches.

  • ✓ Real-time key press visualisation
  • ✓ Input latency measurement (ms)
  • ✓ Stuck key detection
  • ✓ Full keyboard layout display
Keyboard.html Keyboard.js
02

Mouse Tester

Track movement accuracy, click response time, and scroll precision on a live canvas. Detects double-click issues and visualises your cursor trail to expose jitter or sensor problems.

  • ✓ Live cursor tracking canvas
  • ✓ Click response time (ms)
  • ✓ Scroll wheel precision test
  • ✓ Jitter & sensor visualisation
Mouse.html Mouse.js
03

Controller Tester

Plug in any controller and see every button, trigger, and stick respond on a visual layout. Supports Xbox, PS4 & PS5. Auto-detects the device and displays stick drift in real time.

  • ✓ Xbox / PS4 / PS5 support
  • ✓ Stick drift detection
  • ✓ Trigger pressure monitoring
  • ✓ Rumble / vibration test
Controller.html Controller.js
04

Audio Tester

Isolate left and right audio channels to detect balance issues or blown drivers. Play a test tone and check microphone input levels. Works with headsets, speakers, and any audio device.

  • ✓ L / R channel isolation test
  • ✓ Test tone playback
  • ✓ Microphone input monitoring
  • ✓ Volume level visualiser
audio.html audio.js
05

Webcam Tester

Instantly preview your webcam feed in the browser with no drivers or plugins needed. Verify resolution, frame rate, and image quality before any meeting or streaming session.

  • ✓ Live video preview
  • ✓ Resolution & FPS display
  • ✓ No plugins required
  • ✓ Multi-camera selection
Webcam.html Webcam.js
06

Settings

Customise the testing experience. Adjust sensitivity thresholds, toggle visual effects, select input devices, and configure latency display preferences to match your workflow.

  • ✓ Input device selection
  • ✓ Sensitivity thresholds
  • ✓ Visual effects toggle
  • ✓ Latency display options
Setting.html Setting.js
